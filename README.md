# WOODCRAFT2.0

Woodcraft Creations is a handcrafted website that showcases woodworking projects, services, and a portfolio. This repository contains the multi-page site with separate assets for each page (HTML, CSS, JavaScript, images, and documents).

## Pages

- `index.html` — Home
- `about.html` — About the studio and maker
- `services.html` — Services offered
- `portfolio.html` — Project gallery
- `contact.html` — Contact form
- `enquiry.html` — Service-specific enquiries

## Styling & Scripts

- CSS files are located in the `CSS/` folder (one per page).
- JavaScript files are located in the `Javascript/` folder (one per page), providing interactivity such as lightbox gallery, filtering, form validation, and smooth scrolling.

## Project Highlights

- Responsive layout using Flexbox and Grid
- Portfolio gallery with lightbox and category filtering
- Client-side form validation with accessible feedback
- SEO-friendly meta tags, Open Graph, and Schema.org markup
- Performance optimizations including lazy-loading images
